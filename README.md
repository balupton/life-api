# life-api

life-api is a simple Hoodie app... @todo

## Setup

```bash
git clone https://github.com/janecakemaster/life-api.git
cd life-api
npm install
npm run build # default gulp task
```

Start server with

```
npm start
```

## Tests

Install devDependencies by running `npm install` without `--production`

```
npm install
```

Then run tests with

```
npm test
```
